# 📊 Sales Performance Dashboard

## 🎯 Objective
The goal of this project was to analyze over 50,000 sales records to uncover revenue trends, identify top-performing products, and understand customer purchasing behavior.

## 🛠️ Tech Stack
- **Database:** SQL (Window Functions, Aggregations)
- **Data Visualization:** Microsoft Power BI (DAX, Data Modeling)

## 💡 Business Impact
- Executed complex SQL queries to extract and clean sales data.
- Developed an interactive Power BI dashboard featuring dynamic filters.
- Enabled stakeholders to track revenue and profit KPIs, identifying a projected 15% increase in Q3 sales opportunities.
